# gh-attach homebrew tap

Read the source code at [gh-attach](https://github.com/sudosubin/gh-attach).

## Use

To add tap and install gh-attach, run:

```sh
brew tap sudosubin/gh-attach
brew install --cask gh-attach
```

Or one-line install:

```sh
brew install --cask sudosubin/gh-attach/gh-attach
```
